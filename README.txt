
Élan Vert - site HTML/CSS pour hébergement gratuit
Files:
- index.html
- shop.html
- about.html
- contact.html
- styles.css
- assets/ (logo.png and hero.jpg placeholders)

Instructions rapides de déploiement (Netlify):
1. Allez sur https://app.netlify.com/drop
2. Glissez-déposez le dossier 'elanvert_site' (ou le zip) sur la page pour déployer.
3. Netlify générera une URL comme https://elanvert.netlify.app (ou similaire). Pour avoir elanvert.netlify.app exact, connectez votre compte Netlify et personnalisez le site ou utilisez un domaine personnalisé.
4. Remplacez les liens '#' des boutons 'Acheter' par vos URLs produits Gelato.

Remarques:
- Les images actuelles sont des placeholders; remplacez assets/hero.jpg et assets/logo.png par vos visuels finaux en conservant les mêmes noms.
- Pour lier à Gelato, créer des produits sur Gelato, obtenir l'URL produit et coller dans les boutons 'Acheter' sur shop.html.
